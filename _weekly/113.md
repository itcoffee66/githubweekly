## 1. warp 

- 项目名称：warp  – AI 终端工具
- 链接：https://github.com/warpdotdev/warp

Warp 现在是一个智能体开发环境，而一开始他是终端工具。 

Warp其实发布的时间不短了，早在2021年7月，它就以Fast, Rust-based terminal的口号发布了，当时产品就以非常前卫的设计吸引了不少人的关注，而后也第一时间引入AI的功能，并且一直都做的很好，Warp 内置了编程智能体，也可以接入您自己的 CLI 智能体比如Claude Code、Codex、Gemini CLI 等。但是伴随产品发展一直被社区呼吁的就是开源，那就在这周，它终于开源了，并且热度爆表，目前已经达到了40K+ star。

Warp使用时如果要使用AI功能需要登录，一开始你就可以选择是使用terminal模式还是内置agent模式，。如果是agent模式的话，其实就相当于你直接就已经进入了claude code的状态。因为它本身就是一款终端工具，所以在使用上确实会更流畅，更有优势。所以如果你是一个开发者的话，并且你喜欢cli的工作模式，那warp会是一个值得尝试的方案。

另外很让大家关注的是 OpenAI 是项目的 founding sponsor。这一方面说明了产品的水平，同时也让人联想是不是将来会被收编。

Warp 现在已经不只是一个终端了，它还推出了 **Oz，**云端的 Agent 编排平台，可以同时跑最多 40 个并发 Agent，正在进一步完善自己的生态体系。

最后还想说一句，那为什么现在才开源呢？用创始人的说法：开发的瓶颈已经转移了。

以前最值钱的是代码，但现在代码不值钱AI写得飞快，真正重要的是人类做的事情：**定义产品规格、验证行为是否正确**。我觉得这个认知的转变值得所有人去仔细体会，未来的趋势真的在变化。

## 2. hackingtool

- 项目名称：hackingtool – 一站式黑客工具箱
- 链接：https://github.com/Z4nzu/hackingtool

hackingtool是网络安全领域的一把瑞士军刀，它做的事情很直接——把渗透测试各阶段需要的工具整合到一个图形化界面里。

支持的功能包括信息收集（子域名枚举、端口扫描）、漏洞扫描、Web 攻击（XSS、SQL 注入工具集）、无线攻击、密码破解、逆向工程等等，基本覆盖了安全测试的全流程。

它基于 Kali Linux 运行，Python 写的，MIT 协议。最大的优点就是界面友好，不用记几十个命令行工具，点点鼠标就能搞定。对于刚入门安全方向的朋友来说，这个上手成本非常低。

当然还是注意，这东西适合学习和小规模测试用，对于网络安全我建议大家好好学习，不干坏事。

## 3. Skills

- 项目名称：Skills – 真实工程师的skill
- GitHub 链接：https://github.com/mattpocock/skills

本项目作者是Matt Pocock，一个TypeScript 社区的顶流博主，这回他把自己日常用的 Claude Code 技能集全部开源了。不到 3 个月，获得45000+ star，这个增速在开源圈算得上非常快的了。

他特意强调了一点：这不是教你vibe coding的，是给**真正工程开发的人**用的。里面 18+ 个技能模块，涵盖任务拆解、架构设计、TDD 工作流、Git 安全操作、技术文档写作等等。

安装也很简单，一行命令 npx skills@latest add 就行。每个技能都是独立的，你可以按需选装，不需要全装。安全评分拿到了满分 100 分。

项目的介绍里写了很多项目的思路，挺值得阅读一下，比如软件开发中最常见的故障模式是理解偏差，对于 AI agent也是一样的。

今天我们能看到有好多个skill的仓库爆火，从中也能看出AI发展趋势的逐渐变化。

## 4. awesome-codex-skills

- 项目名称：awesome-codex-skills – Codex 技能生态精选
- GitHub 链接：https://github.com/ComposioHQ/awesome-codex-skills

上一个项目是 Matt Pocock 个人的技能实践，这个项目则是社区层面的总结。Composio 团队维护了一个 Codex 技能的精选列表，目标是让 OpenAI 的 Codex CLI 能连接 1000+ 外部应用。

可以做什么呢？比如自动部署、CI/CD 流程自动化、代码审查、连接 Slack/GitHub/Jira 等外部服务——基本上把你日常需要手动操作的工作流，交给 AI 自动跑。每个技能都是模块化的，独立安装按需组合。

大家可能经常听到分享claudecode的东西，而忽略了codex，其实最近codex的更新，还有GPT的模型能力提升都非常的不错，在累加上对国服相对友好和用量充足，在我看来绝对是近期最值得使用的开发方案。

顺便给大家一个直观对比，这几天我正好试用deepseek，我让它开发一个俄罗斯方块的iso应用，那同时也在codex给出了同样的要求。deepseek用了30多分钟，代码给我写了，但是没做ios项目的初始化，让我自己去弄。。。。而codex里，8分钟就开发出了初版，等待deepseek结果的空档我已经做了几轮细节调整，差距很直观。

## 5. Pixelle-Video

- 项目名称：Pixelle-Video – AI 自动生成短视频
- GitHub 链接：https://github.com/AIDC-AI/Pixelle-Video

本项目是一个比较实用的AI工具，它的卖点就是：**输入一个主题，输出一个完整的短视频**。

整个流程是全自动的：

先让 AI 写文案 → 再根据文案生成配图 → 逐帧处理成视频 → 自动合成语音解说 → 匹配背景音乐 → 最后合成成品

每个环节都可以自定义，比如换不同的 AI 模型生成图片、换不同的语音引擎、调整视觉风格等等。模块化设计，灵活性不错。

因为我个人也经常玩一些AI生图和视频生成，其实除了工具，模型的能力还是非常关键，还有就是好的模型成本很高，比如seedance2.0能力很强，但是生成视频的价格也还是蛮贵的。

其实看样子就能感到，这个项目不是很复杂，我猜可能是vibecoding的产物。但是他确实在做一个比较有意思和实用的事情。我也是想给大家做个参考，大家可以作为学习，其实有兴趣完全也可以自己vibecoding出类似的工具，如果买了coding plan的话，那可以去试着搞搞。
