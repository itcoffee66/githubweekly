## 1. DeepSeek-TUI 

- 项目名称：DeepSeek-TUI  – DeepSeek V4的编程 Agent
- 链接：[https://github.com/Hmbown/DeepSeek-TUI](https://github.com/Hmbown/DeepSeek-TUI)

本项目是一个为 DeepSeek v4 量身打造的编程 Agent，它的名字叫DeepSeek-TUI，但它不是 DeepSeek 官方产品，而是个人开发者基于 DeepSeek V4 开发的。

而且非常有意思的是，它不是中国人开发的，而是一个叫 Hunter Bown的美国老哥做的。那他当时在Twitter上宣传自己项目的时候，还特意把推文翻译成了中文，都是为了鲸鱼兄弟，他还求助鲸鱼兄弟们帮他搞一个微信。

当然这个项目爆火一方面是鲸鱼兄弟们的助推，但另一方面，DeepSeek确实也一直缺一个对应的coding agent。gpt+codex，claude+claudecode，包括国内的模型也都有什么qwencode，kimicode，Kcode，所以当一个还不错的项目填上这个空缺的时候，获得热度也是一种必然。

那DeepSeek-TUI也是提供了多种安装方式，npm，cargo，brew，docker都可以，一般来说你能装得明白ClaudeCode，就能搞定它。

**它的 Auto Mode 比较有意思，**会先用 deepseek-v4-flash 判断任务该用哪个模型，  简单任务用deepseek-v4-flash省钱，复杂任务再调用更强deepseek-v4-pro模型。

其实抛开deepseek的光环，我觉得项目中规中矩，其实没什么特别的亮点。在项目的issue中也能看到，现在还有很多问题需要解决，所以大家也不要盲目追捧，还需要时间发展和验证。

## 2. financial-services

- 项目名称：financial-services – Anthropic的金融领域Agent集合
- 链接：[https://github.com/anthropics/financial-services](https://github.com/anthropics/financial-services)

Financial Services，看名字就知道这是Anthropic最近开放出来金融领域Agent 集合，可以用于我们最常看到的金融服务工作流程——投资银行、股票研究、私募股权和财富管理。

金融行业有大量**可结构化认知劳动**——需要理解上下文，而且不是创意工作。Anthropic就是看到这类工作恰好是 Agent 最擅长的区间。

项目里面覆盖的场景非常典型，像投行里的推销Agent、会议准备Agent，研究里的市场研究、财报评审，还有基金管理、财务对账、KYC 审核。能看出来Anthropic 不满足于只做大模型，现在也开始直接下场做“行业工作方法”。

做个形象的比喻，以前你买的是一个 Excel，现在人家直接给你一整套投行分析师常用模板，还顺手把数据连接器和工作流也接好了。

项目里的agent会包含一份 prompt、skill 和 connector，你可以在Claude Cowork里安装或者是走Claude Managed Agents API。

不过话说回来，这个项目门槛其实不低。普通开发者拿来直接玩，可能没那么爽，因为里面很多价值都建立在真实金融数据、内部流程之上的。

## 3. DocuSeal

- 项目名称：DocuSeal – 开源版电子签
- GitHub 链接：[https://github.com/docusealco/docuseal](https://github.com/docusealco/docuseal)

DocuSeal 是一个开源平台，提供安全高效的数字文档签名和处理，它对标的就是传统电子签平台DocuSign，一句话说就是：如果你一直嫌 DocuSign 太贵、太重、太企业味，那可以看看它。 它做的事情很直接，就是在线创建、填写、签署 PDF 文档，而且支持多提交人、邮件通知、存储到 S3 / Azure / Google Cloud、API / Webhook 集成，还能自部署。

安装的话可以直接使用项目的docker镜像，或者也可以去试用在线平台来了解功能。它提供了公司徽标和白标、用户角色、自动提醒、通过短信邀请和身份验证等专业功能。

其实很多公司真正需要的不是花里胡哨的合同平台，只是想把“发合同、填字段、签字、归档”这件事顺利跑完。DocuSeal 的价值就是简化流程，保留关键能力。尤其对很多中小团队来说，这种体验就已经很够用了。 

市面上的电子签产品是不少的，感觉这是一个比较实用的方向。

## 4. easy-vibe

- 项目名称：easy-vibe – 给新手做的 vibe coding 入门课
- GitHub 链接：[https://github.com/datawhalechina/easy-vibe](https://github.com/datawhalechina/easy-vibe)

本项目是datawhale社区出品的一个vibe coding入门课程，它不是在教你某个框架怎么写，而是在教你 怎么和 AI 一起把应用做出来。说白了，这不是传统编程教程，更像一份“会说话就能开始做应用”的学习地图。

仓库里最强的地方是它把很多本来很抽象的东西做得特别可视化。比如 step-by-step 教程、模拟 IDE 交互、RAG 流程演示、终端概念可视化，甚至连 AI 生成图片这种东西都做了动画讲解。你可以把它理解成一个“编程新手友好版的互动教材”，不是让你硬啃概念，而是尽量让你边看边点边试。

这类项目现在还是很有价值。因为很多人嘴上都在说 vibe coding，但现实是，真正的新手上来连环境、IDE、终端、Git、Prompt 都搞不明白。结果就是一直还卡在第一步。

如果你身边有人想学 AI 编程又不知道从哪下手，我觉得这个仓库比很多长篇大论的课程更适合入门。

## 5. SuperSplat

- 项目名称：SuperSplat – 3D Gaussian Splat 在线编辑器
- GitHub 链接：[https://github.com/playcanvas/supersplat](https://github.com/playcanvas/supersplat)

本项目是一个3D 高斯 Splat 编辑器，它做的事情很简单：在浏览器里直接查看、编辑、优化和发布 3D Gaussian Splats。 不用下载安装，打开网页就能开始玩。

Gaussian Splat 是什么呢？可以先把它理解成一种“把现实世界拍成一堆高质量光点，再拼成 3D 场景”的技术。它看起来有点像点云，又有点像体素，但视觉效果通常更真实，成本又比传统高模低一些。SuperSplat 就像这个领域的“在线 PS”，专门拿来处理这些 3D splat 资产。

项目本身是 PlayCanvas 团队做的，走的是很典型的 Web 技术路线。它的优势也正好在这：浏览器即开即用，适合演示、迭代、分享，也适合快速试错。你不一定非得是 3D 开发者，哪怕只是对新型 3D 内容形态感兴趣，进去看看也挺有意思。
